# Icon

该目录中的图标主要整理自 [Koolson/Qure](https://github.com/Koolson/Qure) 项目，原项目是一套面向 Quantumult X 策略组的图标资源。

在保留原始图标风格的基础上，本仓库开发者按自己的使用场景做了继续维护，包括：

- 补充了部分上游没有覆盖的图标
- 调整了部分文件名和目录结构，便于统一管理
- 维护了当前仓库可直接使用的彩色图标索引

## 当前结构

- `Color/`：当前实际使用的彩色图标目录
- `color.json`：与 `Color/` 目录同步的图标索引

## 使用方式

如果需要直接引用本仓库中的图标，可使用：

```text
https://raw.githubusercontent.com/wangyuyan666/Doumu/main/Icon/Color/<图标文件名>
```

例如：

```text
https://raw.githubusercontent.com/wangyuyan666/Doumu/main/Icon/Color/YouTube.png
```

## 说明

- 图标原始设计来源请以 `Koolson/Qure` 为主
- 本仓库中的新增、整理和补充内容由当前仓库开发者维护
- 图标及相关品牌标识仅用于资源整理与学习交流
